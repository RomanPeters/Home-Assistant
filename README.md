# Home Assistant

[Screenshot of the web GUI](example/screenshot.png).

Running on Raspberry Pi 3. 

Other connected hardware includes:
- 2x Milight LED strip RGBW controller
- Milight RGBW bulb
- Milight WW bulb
- Milight WiFi bridge
- Broadlink MP1 WiFi power strip
- Broadlink RM2 Pro
- Apple TV
- Apple HomePod
- Blinkt!
